# demo1

创建基础场景，并展示基础几何体

[文档](https://threejs.org/docs/index.html#manual/zh/introduction/Drawing-lines)

## 做了什么

- 创建场景
  - 添加几何体
    - 添加材质
    - 设置所在位置
  - 添加坐标轴辅助器(用于简单模拟3个坐标轴的对象.红色代表 X 轴. 绿色代表 Y 轴. 蓝色代表 Z 轴.)
- 创建相机
    - 设置所在位置与角度
- 创建渲染器
  - 添加渲染器的 canvas 到 dom 上
  - 渲染场景和相机
- 让几何体动起来
- 浏览器窗口大小改变时更新摄像机和渲染器
- 进入/退出全屏
